# Server in China

This is another piece of the same networking puzzle, but with a specific geographical and functional focus.

The Breakdown img-s.msn.cn: This is a sub-domain for MSN (Microsoft Network). The .cn indicates it is specifically hosted for or routed through China. The img-s prefix usually stands for "image static," meaning this server is responsible for delivering the thumbnails, icons, and photos you see on MSN or Windows widgets.
